console-tetris 是使用 console.log 编写的，在控制台玩的俄罗斯方块。

## 启动

执行以下命令，node版本最好在12以上。

```
npm install
npm start
```
在浏览器打开 http://localhost:8080/

## 操作方式

1.【F12】打开控制台。  
2.按【Enter】开始游戏。  
3.【↑】【↓】【←】【→】控制方向，【Space】控制旋转。

注意，不要让光标焦点落在控制台。
